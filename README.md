# 👔 PoliteLLM: The Corporate Email Translator

PoliteLLM is a "production-ready" microservice that takes your unfiltered, angry workplace rants and translates them into perfectly polished, passive-aggressive corporate speak.

## ✨ Features
- **FastAPI Backend**: Auto-generated OpenAPI/Swagger documentation.
- **LiteLLM Integration**: Seamlessly powered by Google's **Gemma 3 27B IT**.
- **Observability**: Custom callbacks to track latency, token usage, and "Simulated Cost" savings.
- **LLM-as-a-Judge**: Automated evaluation scoring translations on HR Compliance, Meaning Preservation, and "Passive-Aggressiveness" levels.

## 🛠️ Tech Stack
- **LLM:** Gemma 3 27B IT (via Google AI Studio)
- **Framework:** FastAPI / Pydantic
- **Orchestration:** LiteLLM
- **Environment:** Python 3.10+
